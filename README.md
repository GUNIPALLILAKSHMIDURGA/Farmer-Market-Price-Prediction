# Farmer Market Price Prediction

## Project Overview
This project predicts farmer market prices using Machine Learning techniques. The system helps estimate market prices based on different input features.

## Features
- Market price prediction
- Machine Learning model integration
- Frontend interface for user input
- Backend API using Flask

## Technologies Used
- Python
- Machine Learning
- Flask
- HTML
- CSS
- JavaScript
- Joblib

## Project Structure
- `app.py` - Main backend application
- `model/` - Trained ML model files
- `backend/` - Frontend and backend files
- `Farmer_Mar.ipynb` - Model training notebook

## How to Run
1. Install requirements:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
python app.py
```

3. Open browser:
```bash
http://127.0.0.1:5000
```
